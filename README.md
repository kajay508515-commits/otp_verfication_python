**OTP Verification using Python**

Description :
      This project is a simple OTP (One-Time Password) verification system written in Python.
      It generates a random 4-digit OTP and verifies it based on user input.

Features :
      Generates a random 4-digit OTP
      Accepts a 10-digit mobile number
      Verifies OTP entered by the user
      Keeps asking until correct OTP is entered

Requirements :
      Python installed
      NumPy library

Install NumPy using:
      pip install numpy

How to Run :
      Save the code as otp_verification.py
      Open terminal or command prompt
      Run the file:
      python otp_verification.py

How It Works:
      User enters a 10-digit mobile number
      Program generates a random OTP
      User enters the OTP
      If OTP is correct → Verification successful
      If OTP is wrong → User is asked again
          
Example :
      Enter 10 digit mobile number: 9876543210  
      Generated otp: 1234  
      Enter 4 digit otp: 1234  
      otp verified successfully  

Note :
      This project is for learning purposes only.
      The OTP is displayed on the screen and not sent to a real mobile number.
